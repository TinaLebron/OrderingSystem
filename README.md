# OrderingSystem

目的:之前在C#_WindowsForms有作冷飲訂購系統,所以試著嘗試用其它程式語言,像是Java與Swift

冷飲訂購系統(Swift):
https://github.com/TinaLebron/IOS-Swift_BeverageOrderingSystem


功能:  <br>
       • 輸入客戶購買飲料的數量 <br>
       • 數量選擇好按下列印訂購單,會有訊息提示(Detail Inserted) <br>
       • 訂購單會顯示客戶選擇的商品,數量,價錢與當天購買日期 <br>
       
改善: <br>
     • 版面要美化 <br>
