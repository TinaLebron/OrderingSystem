# OrderingSystem

目的:

https://github.com/TinaLebron/IOS-Swift_BeverageOrderingSystem


功能:  <br>
       • 存儲當日所要存的金額 <br>
	     • 首頁會顯示以前到現在所存的加總金額 <br>
	     • 歷史紀錄按鈕會紀錄當下存錢的日期、時間與當下存的金額 <br>
	     • 重置按鈕會跳出提示視窗,此按鈕會將系統所有紀錄清空 <br>
	     • 資料會儲存到SQLite

改善: <br>
     • 版面要美化 <br>
		 • 歷史紀錄可以做刪除 <br>
	   • 功能要增加領錢系統 <br>
